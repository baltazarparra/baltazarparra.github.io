# Project guidance

This repository contains one production site: the Astro portfolio at the
repository root. The legacy Vite/React application was removed.

Use Node.js 22.12+ with pnpm 11.8. The principal commands are:

```bash
pnpm dev
pnpm lint
pnpm check
pnpm build
pnpm launch:check
```

See `AGENTS.md` for the complete architecture, performance, accessibility, and
deployment guidance.
